Таблицы смертности

Таблицы смертности задают вероятность qₓ — вероятность смерти в течение года для застрахованного в возрасте x лет. Они используются во всех актуарных расчётах системы. При отсутствии загруженных таблиц применяется аппроксимация по закону Гомпертца.

Как работает выбор таблицы

Для каждого пола в расчётах используется таблица с наибольшим годом. Если таблиц для нужного пола нет — применяется встроенная аппроксимация Гомпертца (параметры: мужчины a=0,00022 b=0,097; женщины a=0,00015 b=0,090).

Формат CSV для загрузки

CSV-файл должен содержать два столбца с заголовком: 

age,qx
0,0.005320
1,0.000423
2,0.000287
...
80,0.089400

Разделитель — запятая или точка с запятой. Кодировка — UTF-8. Значение qx должно быть в диапазоне (0; 1). Строки с ошибками пропускаются.
